### 1. 기쁨 - 해바라기

| 항목 | 내용 |
|------|------|
| **감정** | 기쁨 |
| **꽃 이름** | 해바라기 |
| **꽃말** | 당신을 보면 행복해요 |
| **이미지 파일명** | `sunflower` |
| **꽃 색상** | 밝은 노랑색, 황금색 |
| **꽃 색상 코드** | #FFD700, #FFA500 |
| **꽃 원산지** | 중앙아메리카 (북미) |
| **꽃 향기** | 약한 풀향기, 거의 무취 |
| **꽃말 유래** | 해바라기가 태양을 향해 고개를 돌리며 성장하는 모습에서 유래. 어린 해바라기는 햇빛을 따라 동서로 움직이며, 이는 '일편단심', '충성심', '동경'을 상징하게 됨. 만개한 후에는 동쪽을 향해 고정되어 아침 햇살을 받으며 곤충을 유인함. |
| **관련 신화/전설** | **잉카 신화**: 고대 잉카 문명에서 해바라기는 태양신 '인티(Inti)'의 상징으로 숭배됨. 잉카의 왕은 태양의 아들로 불렸으며, 해바라기 모양의 순금 왕관과 장신구를 착용. 16세기 스페인 정복자들이 황금빛 해바라기를 실제 황금으로 착각하여 약탈했다는 이야기가 전해지며, 이로 인해 '거짓된 부'라는 꽃말도 생겨남. **그리스 신화**: 물의 요정 클리티에가 태양신 아폴론을 사랑했으나 이루어지지 못하자, 9일 동안 태양만 바라보다가 해바라기로 변했다는 전설 (사실 이는 헬리오트로프라는 다른 꽃의 이야기가 후대에 해바라기와 혼동됨). |

### 2. 행복 - 코스모스

| 항목 | 내용 |
|------|------|
| **감정** | 행복 |
| **꽃 이름** | 코스모스 |
| **꽃말** | 평화로운 사랑 |
| **이미지 파일명** | `cosmos` |
| **꽃 색상** | 연분홍색, 분홍색, 흰색, 빨간색 |
| **꽃 색상 코드** | #FFB6C1 (연분홍), #FF69B4 (분홍), #FFFFFF (흰색), #DC143C (빨강) |
| **꽃 원산지** | 멕시코 (열대 아메리카) |
| **꽃 향기** | 거의 무취, 매우 약한 풀향기 |
| **꽃말 유래** | '코스모스(Cosmos)'는 그리스어 'Kosmos(질서, 조화, 아름다움)'에서 유래. 8개의 꽃잎이 가지런히 질서있게 자리잡고 있는 모습에서 이름을 얻음. 가을바람에 살랑살랑 흔들리는 모습이 마치 소녀가 수줍어하는 것처럼 보여 '소녀의 순정', '순결'이라는 꽃말을 갖게 됨. 18세기 후반 스페인 식물학자 안토니오 호세 카바닐레스 신부가 멕시코에서 발견한 씨앗을 가져와 '가늘게 갈라진 잎을 가진 아름다운 꽃(Cosmos bipinnatus)'이라고 명명함. |
| **관련 신화/전설** | **신의 첫 작품 전설**: 신이 세상을 창조할 때 가장 먼저 만든 꽃이 코스모스였다는 전설. 처음 만든 작품이라 다양한 색깔과 모양을 시도하다 보니 여러 색상의 코스모스가 생겨났고, 이후 다른 꽃들을 만들게 되었다는 이야기. **연인의 전설**: 가난한 나무꾼 요시미와 부잣집 딸 소녀가 사랑에 빠졌으나, 소녀의 부모가 반대하여 결혼을 강요당함. 소녀는 정절을 지키기 위해 분홍색 꽃으로 변했고, 요시미도 그녀를 따라 흰 꽃으로 변했다는 슬픈 전설. 이 두 사람이 변한 꽃이 코스모스라고 전해짐. |

### 3. 감사 - 핑크 장미

| 항목 | 내용 |
|------|------|
| **감정** | 감사 |
| **꽃 이름** | 핑크 장미 |
| **꽃말** | 감사, 존경 |
| **이미지 파일명** | `pink-rose` |
| **꽃 색상** | 연한 분홍색, 진한 분홍색 |
| **꽃 색상 코드** | #FFB6C1, #FF69B4, #FFC0CB |
| **꽃 원산지** | 중동 (페르시아, 메소포타미아) |
| **꽃 향기** | 달콤하고 우아한 장미 향, 중간~강한 향기 |
| **꽃말 유래** | 빅토리아 시대에 꽃으로 감정을 전달하는 '꽃의 언어(Language of Flowers)'가 유행하면서 장미 색상마다 다른 의미가 부여됨. 핑크 장미는 빨간 장미의 열정적인 사랑과 흰 장미의 순수함 사이에 위치하여 '우아한 사랑', '감사', '존경'의 의미를 갖게 됨. 연한 핑크는 부드러운 감탄과 기쁨을, 진한 핑크는 깊은 감사와 인정을 의미함. |
| **관련 신화/전설** | **그리스 신화**: 미의 여신 아프로디테가 사랑하는 아도니스가 멧돼지에게 죽자, 슬퍼하며 달려가던 중 장미 가시에 찔려 피가 흐름. 그 피가 흰 장미에 떨어져 붉은 장미가 되었다는 전설. 핑크 장미는 그 중간 색으로 사랑과 순수함의 조화를 상징. **페르시아 전설**: 나이팅게일(불불새)이 흰 장미를 너무 사랑한 나머지 가슴을 가시에 찔러 피를 흘렸고, 그 피로 장미가 붉게 물들었다는 이야기. 핑크는 그 사랑의 시작을 의미함. |

| **꽃말 유래** | 독일 식물학자 프리드리히 프리제(Friedrich Freese)의 이름에서 유래. 19세기 초 남아프리카에서 발견되어 유럽에 소개됨. 봄을 알리는 꽃으로 신선하고 순수한 이미지를 가져 '순수한 마음', '천진난만', '우정'의 꽃말을 갖게 됨. 달콤한 향기가 설레는 봄날의 기대감을 불러일으킴. |
| **관련 신화/전설** | 프리지아는 비교적 최근에 발견된 꽃이라 고대 신화는 없으나, 빅토리아 시대에 '7번째 결혼기념일의 꽃'으로 지정되어 순수하고 변치 않는 사랑의 상징이 됨. 남아프리카 원주민들 사이에서는 프리지아가 피면 봄비가 온다는 속설이 있어 '희망'과 '새로운 시작'을 상징함. |

| **꽃 이름** | 노란 튤립 |
| **꽃말** | 성공, 명성 |
| **이미지 파일명** | `yellow-tulip` |
| **꽃 색상** | 밝은 노란색, 황금 노랑색 |
| **꽃 색상 코드** | #FFFF00, #FFD700, #FFEB3B |

| **꽃 향기** | 거의 무취 또는 매우 약한 단내 |
| **꽃말 유래** | 밝은 노란색이 태양과 황금을 연상시켜 '부', '성공', '명성'의 의미를 갖게 됨. 17세기 네덜란드의 '튤립 광풍(Tulip Mania)' 시기에 노란 튤립은 특히 귀했고, 이를 소유한 사람은 부와 성공의 상징으로 여겨짐. 봄을 알리는 화려한 꽃으로 겨울을 이겨낸 승리감과 새로운 시작의 의미도 포함. |
| **관련 신화/전설** | **페르시아 전설**: 왕자 파르하드가 공주 시린을 사랑했으나, 시린이 죽었다는 거짓 소식을 듣고 절벽에서 뛰어내림. 그의 피가 흐른 자리마다 빨간 튤립이 피었고, 그의 눈물이 떨어진 곳에서 노란 튤립이 자랐다는 전설. **네덜란드 전설**: 한 소녀에게 세 명의 기사가 각자 왕관(부), 칼(권력), 황금(명예)을 주며 청혼했으나, 소녀는 결정하지 못하고 꽃의 여신에게 자신을 튤립으로 바꿔달라고 부탁. 튤립은 왕관(꽃), 칼(잎), 황금(노란 색)을 모두 가진 꽃이 됨. |

### 7. 사랑 - 빨간 장미

| 항목 | 내용 |
|------|------|
| **감정** | 사랑 |
| **꽃 이름** | 빨간 장미 |
| **꽃말** | 사랑, 애정 |
| **이미지 파일명** | `red-rose` |
| **꽃 색상** | 진한 빨강색, 심홍색 |
| **꽃 색상 코드** | #FF0000, #DC143C, #8B0000 |
| **꽃 원산지** | 중동 (페르시아) |
| **꽃 향기** | 진하고 달콤한 장미 향, 매우 강함 |
| **꽃말 유래** | 고대 그리스·로마 시대부터 빨간 장미는 사랑의 여신 아프로디테(비너스)와 연결되어 '열정적인 사랑'의 상징이 됨. 중세 시대에는 기사들이 사랑하는 여인에게 빨간 장미를 바치는 전통이 있었고, 장미 전쟁(Wars of the Roses)에서도 왕가의 상징으로 사용됨. 빨간색 자체가 열정, 욕망, 사랑을 의미하는 색이라 가장 직접적인 사랑의 표현이 됨. |
| **관련 신화/전설** | **그리스 신화**: 미의 여신 아프로디테와 사랑의 신 에로스의 꽃. 아프로디테가 사랑하는 아도니스를 구하려다 장미 가시에 찔려 피가 흐르자 흰 장미가 빨갛게 물들었다는 전설. **로마 신화**: 사랑의 여신 비너스가 목욕할 때 주변을 장미로 둘러쌓아 은밀함을 지켰다는 이야기. 이후 '비밀'과 '사랑'의 상징이 됨. **이슬람 전설**: 예언자 무함마드가 승천할 때 흘린 땀방울이 땅에 떨어져 장미가 되었다는 전설. |

| **꽃말 유래** | 그리스어 'leirion'에서 유래하며 '순수', '순결', '재생'을 의미. 기독교에서 성모 마리아의 꽃으로 여겨져 순결과 신성함의 상징이 됨. 백합은 겨울을 지나 봄에 다시 피어나는 특성으로 '부활', '재생', '새로운 시작'의 의미를 가짐. 무기력함에서 벗어나 다시 일어설 수 있다는 희망의 메시지를 담고 있음. 조용하고 단아한 모습이 평화와 위로를 줌. |
| **관련 신화/전설** | **그리스 신화**: 제우스의 아내 헤라의 젖이 하늘에 튀어 은하수가 되고, 땅에 떨어진 방울에서 백합이 피었다는 전설. 백합은 신성함과 순수의 상징. **기독교**: 대천사 가브리엘이 성모 마리아에게 수태고지를 할 때 백합을 들고 나타났다는 전설. 이후 백합은 마리아의 순결을 상징하는 '마돈나 릴리'로 불림. **중국 전설**: 한 여인이 남편을 잃고 슬픔에 빠져 아무것도 하지 못하다가, 어느 날 정원에 핀 백합을 보고 다시 살아갈 힘을 얻었다는 이야기. 백합은 재생과 회복의 상징. |

### 19. 실망 - 노란 수선화

| 항목 | 내용 |

| **꽃 색상** | 밝은 노란색, 황금 노랑 |

| **관련 신화/전설** | **그리스 신화**: 아름다운 청년 나르키소스가 님프 에코의 사랑을 거절하자, 복수의 여신 네메시스가 그를 자기 자신만 사랑하도록 저주함. 나르키소스는 물에 비친 자신의 모습에 반해 그 자리를 떠나지 못하고 죽었고, 그 자리에 수선화가 피어남. 이는 이루어질 수 없는 사랑과 자기애의 상징. **페르세포네 신화**: 페르세포네가 수선화를 꺾다가 지하세계의 신 하데스에게 납치당함. 이후 수선화는 유혹과 위험, 불확실한 운명의 상징이 됨. **중국 전설**: 수선화가 물에 비친 자신의 모습을 보며 자기 아름다움에 도취되어 있다는 이야기로, 겸손하지 못한 사랑을 경계하는 의미. |

- 각 꽃의 상세 정보는 이미지 생성 및 사용자 정보 제공 시 활용 가능
